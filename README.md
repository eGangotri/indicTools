# indicTools

####Transliterator####
Transliterator for Indic Scripts with support for back and forth Conversion between 
* SLP1
* ITRANS
* HK
* Velthuis
* Unicode Devenagari
* IAST


####Soon#####
* Sandhi Engine
* Pratyahara Decoder
